# Tarot Card Reading

A mystical tarot card drawing experience built with Next.js.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Draw from the 22 Major Arcana cards
- Card flip reveal animation
- Upright & reversed interpretations
- Mystical dark theme with starry background
